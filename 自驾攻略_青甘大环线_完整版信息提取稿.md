# 青甘大环线 2025 合集完整版信息提取稿

说明：本稿基于 B 站合集 `青甘大环线2025最新攻略` 的 17 条 `ai-zh` 字幕清洗整理。  
目标不是直接输出路书，而是把每条视频里“有执行价值的信息”尽量完整保留下来，同时标出：

- 哪些信息和其他视频重复
- 哪些是推广/工具介绍，已不纳入执行结论
- 哪些信息存在字幕歧义或现场变动风险

## 总体交叉结论

17 条视频反复交叉后，能稳定确认的总信息有：

- 推荐总时长：`9 天` 是主推版本，`8-10 天` 也成立。
- 推荐方向：`逆时针`，原因包括：
  - 先看门源油菜花
  - 旺季更容易避开主流人流节奏
  - 海拔爬升更平缓
- 推荐起点：`西宁`
- 替代起点：`兰州`，但旺季租车价格可能更高；`白银` 可能更便宜
- 推荐主线骨架：
  - `西宁 -> 祁连 -> 张掖 -> 嘉峪关 -> 敦煌 -> 阿克塞 -> 西台/大柴旦 -> 茶卡/乌兰 -> 青海湖 -> 西宁`
- 主线对车型要求不高：`轿车可跑`
- 支线对车型有要求：`二尕公路、天漠公路、部分非铺装段` 更适合高底盘车
- 2025 明确变化：
  - `东台吉乃尔湖` 已基本没水
  - `西台吉乃尔湖` 水也明显变少
  - `U 型公路` 不能安全停车拍照
  - 部分火星地貌类景点开始收费
  - 冷湖到小柴旦更推荐 `215 国道 + 南八仙雅丹`
