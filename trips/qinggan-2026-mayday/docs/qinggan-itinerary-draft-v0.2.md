# 青甘五一行程草案 v0.2

## 当前定下来的整体骨架

- `D1` 西宁 -> 盘大公路 -> 天默公路 -> 祁连
- `D2` 祁连 -> 嘉峪关
- `D3` 嘉峪关 -> 敦煌
- `D4` 敦煌全天：莫高窟 + 鸣沙山月牙泉
- `D5` 敦煌 -> 敦煌周边单点 -> 阿克塞
- `D6` 阿克塞 -> 火星一号公路 -> 水上雅丹
- `D7` 水上雅丹 -> 大柴旦
- `D8` 大柴旦 -> 茶卡
- `D9` 茶卡 -> 青海湖 -> 西宁机场

## 目前各天的想法

### D1

西宁出发，走盘大公路和天默公路到祁连。  
这一天以公路体验为主，不安排正式景区。  
预计公里数：`300-360km`  
预计耗时：`6-8h`  
中间只考虑轻量短停，比如：

- 达坂山观景台
- 门源 / 青石嘴补给短停

### D2

祁连正常导航到嘉峪关。  
这一天不额外讨论特殊路线，也不安排中途正式景点。  
主要就是推进到嘉峪关，路上只做必要补给和短停。  
预计公里数：`420-520km`  
预计耗时：`6.5-8.5h`  
嘉峪关景区当前先不考虑。

### D3

默认先写成：

`嘉峪关 -> 大地之子 -> 敦煌`

预计公里数：`380-450km`  
预计耗时：`5-6.5h`

另外也保留两个可选替换项，后面再看要不要改：

1. 榆林窟
2. 锁阳城

### D4

敦煌全天固定为：

- 莫高窟
- 鸣沙山月牙泉

预计公里数：`40-80km`  
预计耗时：`1-2h`

这一天先不再叠加别的敦煌周边点。

### D5

这一天定位是轻转场 + 单点。  
先保留两条方案，后面再决定：

- `A` 敦煌 -> 阳关 -> 阿克塞
- `B` 敦煌 -> 玉门关 -> 阿克塞

对应估算：

- `D5A` 预计公里数：`180-260km`
- `D5A` 预计耗时：`3-5h`
- `D5B` 预计公里数：`220-320km`
- `D5B` 预计耗时：`4-6h`

现在先不定死，不影响整体行程框架。

### D6

`阿克塞 -> 火星一号公路 -> 水上雅丹`

这一天明确会经过火星一号公路。  
整体还是以荒漠公路转场和地貌体验为主，不再额外塞重景点。  
预计公里数：`420-520km`  
预计耗时：`7-9h`  
目前倾向住水上雅丹附近，这样可以把后半段行程拆开，整体没那么累。

### D7

`水上雅丹 -> 大柴旦`

这一天目前保留两种路线，暂时不做最后决定。

#### 方案 A：315 主线稳定版

- 预计公里数：`200-260km`
- 预计耗时：`3-4.5h`
- 路线更直接
- 更适合把这一天当作回收日
- 有典型荒漠公路氛围
- 如果走 `315`，可以理解为会进入大家常说的 `U 型公路` 那类景观语境
- 但正式攻略里不会写成停车打卡点，只会写成沿途车览

#### 方案 B：215 / 南八仙景观增强版

- 预计公里数：`220-300km`
- 预计耗时：`4-5.5h`
- 更强调雅丹、无人区、公路景观体验
- 更像继续玩一条风景路
- 但这一天也会更偏景观日，整体体感不如方案 A 轻

### D8

`大柴旦 -> 茶卡`

这一天的主任务是推进到茶卡。  
茶卡盐湖目前不设成必须完成的明确主点。  
预计公里数：`430-520km`  
预计耗时：`6.5-8h`  
更倾向于到达后再根据时间、天气、体力决定要不要进去。  
如果进去，也按轻量版处理。

### D9

`茶卡 -> 青海湖 -> 西宁机场`

这一天会保留青海湖。  
如果条件合适，也想体验一下短时骑行，但只按轻体验来安排。  
预计公里数：`320-420km`  
预计耗时：`5-7h`  
目前原则是：

- 看湖为主，骑行为辅
- 骑行控制在 `1-1.5 小时`
- 优先选成熟景区型点位
- 不环湖，不绕行，不额外加别的点
- 这一天还要兼顾晚上 `19:30` 从西宁起飞

## 目前还没完全定下来的点

1. `D3` 最终是保大地之子，还是改成榆林窟 / 锁阳城
2. `D5` 最终选阳关还是玉门关
3. `D7` 最终走 `315` 主线还是 `215 / 南八仙`
4. `D9` 青海湖最终落哪个具体点位
5. `D9` 骑行最后是写进正式版，还是保留成现场可选项

## 整体思路

前半段先把主线跑顺，敦煌固定两大核心，后半段控制强度，最后保住青海湖和返程。
