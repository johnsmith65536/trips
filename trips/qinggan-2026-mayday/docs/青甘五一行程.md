# 青甘五一行程

## 行程总览

- `D0` `2026-04-25` 深圳 -> 西宁，西宁市区游玩
- `D1` `2026-04-26` 西宁 -> 盘大公路 -> 天默公路 -> 祁连
- `D2` `2026-04-27` 祁连 -> 祁连 9 号公路 -> 嘉峪关
- `D3` `2026-04-28` 嘉峪关 -> 敦煌
- `D4` `2026-04-29` 敦煌全天：莫高窟 + 鸣沙山月牙泉
- `D5` `2026-04-30` 敦煌博物馆 -> 阳关 / 玉门关 -> 阿克塞
- `D6` `2026-05-01` 阿克塞 -> 火星一号公路 -> 水上雅丹
- `D7` `2026-05-02` 水上雅丹 -> 大柴旦
- `D8` `2026-05-03` 大柴旦 -> 茶卡
- `D9` `2026-05-04` 茶卡 -> 青海湖 -> 西宁机场

## 整体执行策略

这次线路的主线很清楚：前半段先把转场跑顺，把敦煌作为固定核心；中段把瓜州和敦煌周边点控制在可执行范围内；后半段按景观回收 + 稳定返程来收口。

先看全程节奏：

- 全程最轻松的日子是 `D0` 和 `D4`，适合恢复体力、轻松游玩
- 全程最硬的日子是 `D2`、`D6`、`D8`，这三天不适合临时加正式景点
- `D3` 弹性最大，但前提是先选定版本，不要当天现场混搭
- `D5` 虽然整体不算重，但已经是 `博物馆 + 单点遗址` 的结构，不再额外加第三个主项
- `D7` 是后半段的景观回收日，重点是保住 `东台吉乃尔湖 / 翡翠湖 / U 型公路沿线`
- `D9` 是全程最敏感的返程日，青海湖体验要完全服从航班和机场节奏

需要重点早出发的日期：

- `D1` 尽量早出发，把好看的山口和公路段放在白天
- `D2` 以推进为主，早出发能显著减轻傍晚疲劳
- `D3` 若选 `榆林窟主项版` 或 `双文化重点版`，建议按早出发处理
- `D6` 必须按高负荷转场日理解，越晚出发，越容易把短停点全部挤没
- `D8` 以稳定到达茶卡为第一优先级，不适合睡到太晚再上路
- `D9` 需要为进机场、还车、安检预留充分缓冲，不能把白天安排得过满

这些项目都可以砍，砍掉也不伤主线：

- `D0` 的 `南山公园 / 城市高点`
- `D3` 的第二文化点
- `D5` 下午遗址的深度游览
- `D6` 的大部分短停点
- `D8` 的茶卡盐湖进入
- `D9` 的青海湖骑行

## 出发前最后确认

这几项不要拖到路上再想，最好在正式出发前先拍板：

1. `D3` 最终选 `榆林窟主项版`、`瓜州文化串联版`，还是 `双文化重点版`
2. `D5` 最终选阳关还是玉门关
3. `D9` 青海湖最终落哪个具体点位
4. `D9` 骑行最后是写进正式版，还是保留成现场可选项

## 每日安排

### D0 `2026-04-25`

路线：`深圳 -> 西宁`

当天定位：

- 到达日 + 适应日
- 以低强度活动为主

驾驶负荷：

- 预计公里数：`20-60km`
- 预计耗时：`0.5-2h`

核心安排：

- 主点：`西宁市博物馆`
- 随后在 `东关片区 / 老城街区` 轻松散步
- 如果当天落地早、状态也不错，可以顺手加 `南山公园` 或城市高点看西宁全景

执行原则：

- `博物馆` 作为主项
- `老城散步` 和 `高点城景` 作为可伸缩项
- 如果落地时间偏晚，那就只留市区轻逛和早点休息

餐饮建议：

- `D0` 以市区本地风味为主
- 正餐优先考虑 `手抓羊肉 / 炕锅羊肉`
- 可搭配 `酿皮 / 甜醅 / 老酸奶` 等轻小吃
- 不建议做高强度夜市暴走，整体以轻松、稳妥、早点休息为主

### D1 `2026-04-26`

路线：`西宁 -> 盘大公路 -> 天默公路 -> 祁连`

当天定位：

- 以公路体验为主
- 不安排正式景区

驾驶负荷：

- 预计公里数：`300-360km`
- 预计耗时：`6-8h`

核心安排：

- 中间只做轻量短停
- 可考虑 `达坂山观景台`
- 可在 `门源 / 青石嘴` 做补给短停

执行原则：

- 不为了单个点专门绕路
- 把重点放在公路观感和稳定推进上

餐饮建议：

- `D1` 更适合走公路餐饮思路，早餐在西宁吃扎实
- 午间可优先考虑 `面片 / 牛肉面 / 盖碗茶 + 饼类` 这类出餐快、补能量稳定的组合
- 到祁连后晚餐可以考虑 `牦牛肉 / 黄蘑菇炖鸡 / 土火锅 / 家常炒菜`
- 这一天不建议为了“找一顿特别好的”专门绕路，重点是稳妥补给

### D2 `2026-04-27`

路线：`祁连 -> 祁连 9 号公路 -> 嘉峪关`

当天定位：

- 长转场日
- 以推进到嘉峪关为唯一主任务

驾驶负荷：

- 预计公里数：`420-520km`
- 预计耗时：`6.5-8.5h`

核心安排：

- 途中经过 `祁连 9 号公路`
- 不再额外展开别的路线
- 不安排中途正式景点

执行原则：

- 路上只做必要补给和短停
- 嘉峪关景区这次先不排

餐饮建议：

- `D2` 是长转场日，餐饮以效率优先
- 早餐建议在祁连吃热食，优先 `面食 / 粥 / 鸡蛋 / 包子` 这类耐饿组合
- 午餐按沿线城镇补给解决即可，不建议为了吃专门拉长停留
- 到嘉峪关后晚餐可以转向 `西北面食 + 烤串 + 炒菜` 的组合
- 如果状态不错，可尝试 `烤肉 / 炒炮 / 臊子面 / 驴肉黄面` 这类偏河西走廊口味的东西

### D3 `2026-04-28`

路线：`嘉峪关 -> 敦煌`

当天定位：

- 弹性日
- 不去 `大地之子`
- 只在下面三个版本里选一个

驾驶负荷：

- `A` 预计公里数：`430-520km`，预计耗时：`6.5-8.5h`
- `B` 预计公里数：`400-480km`，预计耗时：`6-8h`
- `C` 预计公里数：`450-560km`，预计耗时：`7-9h`

核心安排：

- `A` 榆林窟主项版：`嘉峪关 -> 榆林窟 -> 瓜州吃瓜 -> 敦煌`
- `A` 以 `榆林窟` 为唯一核心文化主项，逻辑最清楚
- `B` 瓜州文化串联版：`嘉峪关 -> 酒泉桥湾城遗址 -> 玄奘取经博物馆 -> 瓜州吃瓜 -> 敦煌`
- `B` 不押注 `榆林窟`，而是把瓜州文化点轻量串起来
- `C` 双文化重点版：`嘉峪关 -> 榆林窟 -> 桥湾城遗址或玄奘取经博物馆 -> 瓜州吃瓜 -> 敦煌`
- `C` 只允许按 `榆林窟 + 1 个瓜州文化点` 的结构执行

执行原则：

- 临出发前先拍板 `A/B/C`
- 当天不要再临时混搭版本
- `A` 不再叠加 `桥湾城遗址` 和 `玄奘取经博物馆`
- `C` 不建议写成 `榆林窟 + 桥湾城遗址 + 玄奘取经博物馆` 全收
- 如果前一晚休息一般，或当天出发偏晚，就不要选 `C`

餐饮建议：

- `D3` 适合把 `瓜州吃瓜` 写成明确体验项，但它更像点心或路上补给，不是正餐替代
- 如果走 `榆林窟` 方向，建议午餐按简洁快餐处理，避免把正餐拖太晚
- 瓜州一带可留意 `清汤羊肉 / 面食 / 饼类 / 本地瓜果`
- 到敦煌后的晚餐更值得认真吃，可以考虑 `驴肉黄面 / 羊肉类 / 胡羊焖饼 / 杏皮水`
- 若当天文化点较多，晚餐建议以热食为主，不建议只靠零食和水果顶过去

### D4 `2026-04-29`

路线：`敦煌市内`

当天定位：

- 敦煌核心游玩日
- 固定双主项日

驾驶负荷：

- 预计公里数：`40-80km`
- 预计耗时：`1-2h`

核心安排：

- `莫高窟`
- `鸣沙山月牙泉`

执行原则：

- 这一天就不再叠加别的敦煌周边点
- 行程顺序按莫高窟预约场次倒排更稳

餐饮建议：

- `D4` 是敦煌核心游玩日，餐饮可以兼顾体验和节奏
- 早餐建议吃饱，优先 `牛肉面 / 粥 / 鸡蛋 / 饼类`
- 白天景区间穿插时可准备 `酸奶 / 水果 / 坚果 / 能量零食`
- 晚餐可以把敦煌代表性风味集中体验一下，如 `驴肉黄面 / 羊肉 / 浆水面 / 杏皮水`
- 如果鸣沙山安排到傍晚，晚餐可以适当推后，但不要拖太晚

### D5 `2026-04-30`

路线：`敦煌 -> 阳关 / 玉门关 -> 阿克塞`

当天定位：

- 轻转场日
- `博物馆 + 一个遗址点`

驾驶负荷：

- `A` 预计公里数：`180-260km`
- `A` 预计耗时：`3.5-5.5h`
- `B` 预计公里数：`220-320km`
- `B` 预计耗时：`4.5-6.5h`

核心安排：

- 固定项：`敦煌博物馆`
- `A` 敦煌博物馆 -> 阳关 -> 阿克塞
- `B` 敦煌博物馆 -> 玉门关 -> 阿克塞

执行原则：

- `敦煌博物馆` 作为白天固定文化项
- `阳关 / 玉门关` 只选一个，不做双遗址叠加
- 如果上午博物馆停留时间偏长，下午遗址按轻量版处理，不追求完整游览

餐饮建议：

- `D5` 适合把午餐放在敦煌城区解决，选择会更多也更稳
- 中午可以继续补 `敦煌没吃够的代表性东西`，比如 `驴肉黄面 / 羊肉类 / 面食`
- 如果当天走遗址线，下午更适合以 `咖啡 / 茶饮 / 水果 / 小零食` 做轻补给
- 晚上到阿克塞后，餐饮以 `热汤面 / 盖饭 / 炒菜 / 羊肉类` 这种恢复型晚餐更合适

路线：`阿克塞 -> 火星一号公路 -> 水上雅丹`

当天定位：

- 荒漠公路转场和地貌体验日

驾驶负荷：

- 预计公里数：`420-520km`
- 预计耗时：`7-9h`

核心安排：

- `1 个主线体验 + 1 个重点景观点 + 多个沿线短停`
- `火星一号公路`
- `黑独山`
- `博罗转井`
- `当金山口`
- `冷湖石油小镇`
- `双色公路`

执行原则：

- `博罗转井 / 当金山口 / 冷湖石油小镇 / 双色公路` 都按短暂停留处理
- 不把这些短停点写成并列主景点，避免现场节奏失控
- 住水上雅丹附近，这样后半段能拆得更舒服一些

餐饮建议：

- `D6` 是典型高负荷公路日，餐饮关键词是 `提前吃饱 + 车上带够`
- 早餐尽量在阿克塞吃热且顶饱，优先 `面食 / 粥 / 鸡蛋 / 牛羊肉类`
- 白天路上更适合准备 `面包 / 牛肉干 / 坚果 / 巧克力 / 功能饮料 / 咖啡`
- 这一天不要把希望寄托在临时碰到好吃的正餐上，补能量比“吃得好玩得好”更重要
- 到水上雅丹附近后，晚餐以 `热汤 / 热菜 / 主食` 为优先，尽量不要再吃得太随便

### D9 `2026-05-04`

路线：`茶卡 -> 青海湖 -> 西宁机场`

当天定位：

- 返程日

驾驶负荷：

- 预计公里数：`320-420km`
- 预计耗时：`5-7h`

核心安排：

- 保留青海湖
- 如果条件合适，也可以安排一个短时骑行
- 骑行只按轻体验处理

执行原则：

- 看湖为主，骑行为辅
- 骑行控制在 `1-1.5 小时`
- 优先选成熟景区型点位
- 不环湖，不绕行，不额外加别的点
- 返程缓冲不足时，应立即砍掉骑行和额外停留

餐饮建议：

- `D9` 是返程日，餐饮要服从机场节奏，不建议把“吃一顿特别好的”作为目标
- 早餐在茶卡吃饱，优先热食和高碳水
- 青海湖一带如果只做短停，可搭配 `酸奶 / 烤肠 / 饼类 / 简餐` 这种轻量补给
- 如果返西宁时间充裕，离机场较近时再补一顿正式正餐会更稳
- 返程当天尽量少碰太油太刺激的东西，避免影响驾驶和航班状态

### 行李与车载清单

#### 随身行李

- 身份证、驾驶证、行驶相关材料
- 太阳镜、帽子、防晒霜、润唇膏
- 冲锋衣或防风外套、抓绒或薄羽绒
- 保温杯、个人常用药、肠胃药、止痛药
- 纸巾、湿巾、一次性口罩
- 充电器、充电宝、备用数据线

#### 高原与干燥环境相关

- 补水电解质饮料或冲剂
- 润眼液、保湿喷雾可按个人习惯准备
- 如果对高海拔反应比较敏感，可自备便携氧气或按需准备相关药物
- 晚间保暖层不要省，尤其是大柴旦、茶卡一带

#### 车载物资

- 车载手机支架、点烟器或多口车充
- 纸巾、湿巾、垃圾袋
- 瓶装水、功能饮料、咖啡、简单高热量零食
- 备用墨镜、遮阳用品
- 小手电或头灯
- 车载抽纸、消毒湿巾、一次性雨衣可按需带

#### 建议在敦煌或阿克塞前补齐的东西

- `D6-D7` 的饮水和功能饮料
- 可久放的干粮和零食
- 纸巾、湿巾、防晒补给
- 如果拍照较多，补好相机电池和存储卡

## 执行补充

### 补给原则

- `D1-D2` 沿线城镇相对连续，补给压力不大，但仍建议在西宁、祁连、嘉峪关按城区标准补满
- `D3` 如果走 `榆林窟` 方向，午餐和饮水不要拖到太晚，尽量在瓜州方向解决
- `D5` 是敦煌最后一个补给相对从容的白天，建议把零食、饮水、纸巾、车载补给在这一天补齐
- `D6` 是全程最需要前置补给意识的一天，阿克塞出发前建议把油、饮水、功能饮料、干粮一次补足
- `D7` 虽然公里数不算最长，但景观点分散，出发前仍建议在住宿地先完成早餐和补给
- `D8-D9` 进入返程收口阶段，不建议把补给节奏压到路上临时解决

### 每日风险提醒

- `D0` 到达日主要风险是落地偏晚和初到高原体力波动，晚上不要硬撑
- `D1` 风景公路好看，但不值得为了拍照反复停车，整体仍应以平稳推进为主
- `D2` 是典型长转场，后半程容易疲劳，晚到嘉峪关后不建议再安排夜活动
- `D3` 的核心风险不是路，而是版本混搭；一旦定了 `A/B/C`，当天就不要临时加码
- `D4` 固定双核心景点日，主要风险在预约、排队和当天风沙，不在于公里数
- `D5` 虽然相对轻，但如果上午博物馆拖长，下午遗址体验必须做减法
- `D6` 是全程最高风险日之一，风险点在长距离、荒漠路段、节假日车流和体力透支
- `D7` 景观密度高，但不宜把每个点都当成正式停留点，否则会消耗掉 `D8` 的状态
- `D8` 主任务是到达茶卡，任何盐湖体验都排在“稳定到达并休息”之后
- `D9` 所有体验都必须服从 `19:30` 航班，返程缓冲不足时应立即砍掉骑行和额外停留

### 摄影 / 看景窗口

这一段不是摄影攻略，主要是提醒自己哪些地方更适合在哪个时段去看。

#### 更适合白天完成的点

- `D1` 的山口、公路和雪山视野更适合白天完成
- `D6` 的荒漠地貌、公路层次和 `黑独山` 更依赖白天光线与安全驾驶条件
- `D7` 的 `东台吉乃尔湖` 和 `翡翠湖` 更适合白天看湖色
- `D8` 的茶卡盐湖如果要轻量进入，也更适合白天或下午早些时候处理

#### 更适合留出傍晚弹性的点

- `D0` 的城市散步和城市高点可以根据落地时间决定是否保留到傍晚
- `D4` 的鸣沙山月牙泉适合给傍晚留弹性，但不建议为了夜色压缩休息
- `D9` 的青海湖若只是看湖短停，可以根据返程节奏决定是否保留一点傍晚感受，但永远排在航班之后

#### 不值得为拍照硬撑的情况

- `D2` 长转场日傍晚疲劳明显时，不值得为路边景色反复停车
- `D3` 若当天选的是文化版路线，不建议为了“顺便再拍一个点”压缩到敦煌的到达时间
- `D6` 若当天风大、体力下降或到点偏晚，应直接减少短停，不为照片硬拖
- `D8-D9` 已进入返程收口段，拍照优先级应明显低于休息和时间缓冲
