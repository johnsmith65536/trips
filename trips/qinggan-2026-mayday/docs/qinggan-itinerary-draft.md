## 目前各天的想法

### D0 `2026-04-25`

从深圳飞往西宁，当天在西宁市区游玩并过夜。  
这一天定位为到达日 + 适应日，以低强度活动为主。  
预计公里数：`20-60km`  
预计耗时：`0.5-2h`  
当前按城市散步版处理，建议结构为：

- 主点：`西宁市博物馆`
- 随后在 `东关片区 / 老城街区` 轻松散步
- 若当天落地较早且状态不错，可加 `南山公园` 或城市高点看西宁全景

执行原则：

- `博物馆` 作为主项
- `老城散步` 和 `高点城景` 作为可伸缩项
- 若落地时间偏晚，就只保留市区轻逛和早点休息

餐饮建议：

- `D0` 以市区本地风味为主
- 正餐优先考虑 `手抓羊肉 / 炕锅羊肉`
- 可搭配 `酿皮 / 甜醅 / 老酸奶` 等轻小吃
- 不建议做高强度夜市暴走，整体以轻松、稳妥、早点休息为主

### D1 `2026-04-26`

西宁出发，走盘大公路和天默公路到祁连。  
这一天以公路体验为主，不安排正式景区。  
预计公里数：`300-360km`  
预计耗时：`6-8h`  
中间只考虑轻量短停，比如：

- 达坂山观景台
- 门源 / 青石嘴补给短停

### D2 `2026-04-27`

祁连正常导航到嘉峪关。  
这一天不额外讨论特殊路线，也不安排中途正式景点。  
主要就是推进到嘉峪关，路上只做必要补给和短停。  
预计公里数：`420-520km`  
预计耗时：`6.5-8.5h`  
嘉峪关景区当前先不考虑。

### D3 `2026-04-28`

这一天不再考虑 `大地之子`。  
当前保留三个可执行版本，后面从中选一个正式落版。

#### 方案 A：榆林窟主项版

- 路线：`嘉峪关 -> 榆林窟 -> 瓜州吃瓜 -> 敦煌`
- 预计公里数：`430-520km`
- 预计耗时：`6.5-8.5h`
- 这版把 `榆林窟` 作为当天唯一核心文化主项
- 整体逻辑最清楚，适合把 `D3` 定义成单核心景点日
- 为了控制节奏，不再叠加 `桥湾城遗址` 和 `玄奘取经博物馆`

#### 方案 B：瓜州文化串联版

- 路线：`嘉峪关 -> 酒泉桥湾城遗址 -> 玄奘取经博物馆 -> 瓜州吃瓜 -> 敦煌`
- 预计公里数：`400-480km`
- 预计耗时：`6-8h`
- 这版不押注 `榆林窟`，而是把瓜州文化点做轻量串联
- 单点冲击力不如榆林窟，但整体更均衡、更灵活
- 如果希望当天多看两个文化点，这版更稳

#### 方案 C：双文化重点版

- 路线：`嘉峪关 -> 榆林窟 -> 桥湾城遗址或玄奘取经博物馆 -> 瓜州吃瓜 -> 敦煌`
- 预计公里数：`450-560km`
- 预计耗时：`7-9h`
- 这是 `D3` 的上限版，只允许在 `榆林窟 + 1 个瓜州文化点` 的结构内执行
- 不建议写成 `榆林窟 + 桥湾城遗址 + 玄奘取经博物馆` 全收
- 如果前一晚休息一般，或当天出发偏晚，就不建议选这版

### D4 `2026-04-29`

敦煌全天固定为：

- 莫高窟
- 鸣沙山月牙泉

预计公里数：`40-80km`  
预计耗时：`1-2h`

这一天先不再叠加别的敦煌周边点。

### D5 `2026-04-30`

这一天定位是轻转场 + 博物馆 + 单点遗址。  
`敦煌博物馆` 直接纳入固定项，再从 `阳关 / 玉门关` 中二选一。

固定项：

- `敦煌博物馆`

可选方案：

- `A` 敦煌博物馆 -> 阳关 -> 阿克塞
- `B` 敦煌博物馆 -> 玉门关 -> 阿克塞

对应估算：

- `D5A` 预计公里数：`180-260km`
- `D5A` 预计耗时：`3.5-5.5h`
- `D5B` 预计公里数：`220-320km`
- `D5B` 预计耗时：`4.5-6.5h`

执行原则：

- `敦煌博物馆` 作为白天固定文化项
- `阳关 / 玉门关` 只选一个，不做双遗址叠加
- 如果上午博物馆停留时间偏长，下午遗址按轻量版处理，不追求完整游览

### D6 `2026-05-01`

`阿克塞 -> 火星一号公路 -> 水上雅丹`

这一天明确会经过火星一号公路。  
整体还是以荒漠公路转场和地貌体验为主。  
预计公里数：`420-520km`  
预计耗时：`7-9h`  
目前倾向住水上雅丹附近，这样可以把后半段行程拆开，整体没那么累。  
当天按 `1 个主线体验 + 1 个重点景观点 + 多个沿线短停` 来理解更合适。

主线体验：

- `火星一号公路`

重点景观点：

- `黑独山`

沿线短停点：

- `博罗转井`
- `当金山口`
- `冷湖石油小镇`
- `双色公路`

执行原则：

- `火星一号公路` 作为当天绝对主项
- `黑独山` 作为重点停留点
- `博罗转井 / 当金山口 / 冷湖石油小镇 / 双色公路` 都按短暂停留处理
- 不把这些短停点写成并列主景点，避免现场节奏失控

### D7 `2026-05-02`

`水上雅丹 -> 大柴旦`

这一天明确保留以下景观项：

- `东台吉乃尔湖`
- `翡翠湖`
- `U 型公路`

预计公里数：`220-300km`  
预计耗时：`4-5.5h`

这一天可以理解为景观回收日：

- `东台吉乃尔湖` 保留为湖色体验点
- `翡翠湖` 保留为大柴旦方向的明确主点
- `U 型公路` 只按沿途车览或短停处理，不写成正式下车打卡主项
- 整体节奏仍然要控制，避免把这一天玩成第二个高负荷日

### D8 `2026-05-03`

`大柴旦 -> 茶卡`

这一天的主任务是推进到茶卡。  
茶卡盐湖放在这一天处理，但仍然不设成必须完成的明确主点。  
预计公里数：`430-520km`  
预计耗时：`6.5-8h`  
更倾向于到达后再根据时间、天气、体力决定要不要进去。  
如果进去，也按轻量版处理。  
这一天的优先级仍然是先稳妥到达茶卡，再看盐湖。

### D9 `2026-05-04`

`茶卡 -> 青海湖 -> 西宁机场`

这一天会保留青海湖。  
如果条件合适，也想体验一下短时骑行，但只按轻体验来安排。  
预计公里数：`320-420km`  
预计耗时：`5-7h`  
目前原则是：

- 看湖为主，骑行为辅
- 骑行控制在 `1-1.5 小时`
- 优先选成熟景区型点位
- 不环湖，不绕行，不额外加别的点
- 这一天还要兼顾晚上 `19:30` 从西宁起飞
